# Pong Game

In this project i used  (Gorgon Library)[https://github.com/darkgazeorg/Gorgon]  for creating 2D Graphics
You can install the gorgon from the given link


After you installed the gorgon , only thing you need to do is compilation of the project .
Enter the Project root folder and execute : 

    mkdir build
    cd build
    cmake ..
    make

After that commands your executable file will be in ```Bin/``` folder with name ```Pong```
